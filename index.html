Designed for absolute data control.

Vault pairs mathematical client-side proofs with total storage independence.

01

Zero-Knowledge Envelope

Keys are derived client-side from your master password using PBKDF2. Raw secrets never interact with external networks.

02

Supabase Integration

Configure your own public database connection parameters. Maintain total ownership of server backups and keys.

03

Entropy Generator

Generate cryptographically strong tokens client-side. Live entropy calculation tracks credential complexity dynamically.

CRYPTO STANDARD

Native Web Cryptography

Vault relies strictly on native browser Web Crypto implementations. Keys are generated with 100,000 PBKDF2 iterations, exporting a 256-bit AES-GCM envelope.

Read crypto.js implementation

// zero_knowledge_encrypt.js

// Derive key from master password const key = await VaultCrypto.deriveKey(pw, salt); // Encrypt payload client-side const payload = await VaultCrypto.encrypt(secret, key); // Insert into your Supabase database await supabase.from('passwords').insert({ password: payload.ciphertext, iv: payload.iv });
