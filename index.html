Designed for complete privacy.

Vault pairs absolute data portability with mathematical proofs to keep your digital identity secure.

Zero-Knowledge Architecture

Encryption keys are derived client-side from your master password using PBKDF2. Your raw passwords never leave your browser.

Supabase Integration

Bring your own public database storage. Connect your custom database credentials locally to completely control your backups.

Custom Entropy Generator

Generate high-entropy credentials based on length, special symbols, casing, and digits. Evaluated instantly via a live safety tracker.

CRYPTO STANDARD

Open Web Cryptography Standard

Vault relies strictly on native browser Web Crypto implementations. Keys are generated with 100,000 PBKDF2 iterations, exporting a 256-bit AES-GCM envelope.

Read crypto.js implementation

// zero_knowledge_encrypt.js

// Derive key from master password const key = await VaultCrypto.deriveKey(pw, salt); // Encrypt payload client-side const payload = await VaultCrypto.encrypt(secret, key); // Insert into your Supabase database await supabase.from('passwords').insert({ password: payload.ciphertext, iv: payload.iv });
